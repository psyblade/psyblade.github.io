HTML Elements (The DIV element and the IMG element)

1: Create the DIV elements

We insert 3 DIV elements and an IMG element between the 'body' tags of the HTML document.


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Dungeon Chase</title>
<style>
body {
	background-color:purple;
}

.box {
	width:200px;
	height:100px;
	display:inline-block;
}

#box1 {
	background-color:blue;
}

#box2 {
	background-color:red;
}

#box3 {
	background-color:green;
}

#image1 {
	width:200px;
	height:100px;
}
</style>
</head>
<body>

<div id="box1" class="box" ></div>
<div id="box2" class="box" ></div>
<div id="box3" class="box" ></div>
<img id="image1" class="image">

<script>

</script>
</body>
</html>
Hello World

2: Stack DIV elements

We stack the blue and red DIV elements and the IMG element in between and put them all into the green box.


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Dungeon Chase</title>
<style>
body {
	background-color:purple;
}

.box {
	width:200px;
	height:100px;
	display:inline-block;
}

#box1 {
	width:200px;
	height:100px;
	background-color:blue;
}

#box2 {
	width:200px;
	height:100px;
	background-color:red;
}

#box3 {
	width:300px;
	height:400px;
	background-color:green;
}

#image1 {
	width:200px;
	height:100px;
	opacity:0.5;
}
</style>
</head>
<body>

<div id="box3" class="box" >
	<div id="box1" class="box" >Hello World</div>
	<img id="image1" class="image">
	<div id="box2" class="box" ></div>
</div>

<script>
</script>
</body>
</html>
Hello World

3: Layer DIV elements with z-index

We arrange the blue DIV on top of the IMG element and the IMG element on top of the red DIV


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Dungeon Chase</title>
<style>
body {
	background-color:purple;
}

.box {
	width:200px;
	height:100px;
	display:inline-block;
}

#box1 {
	position:absolute;
	top:50px;
	left:50px;
	z-index:3;
	width:200px;
	height:100px;
	background-color:blue;
}

#image1 {
	position:absolute;
	top:100px;
	left:100px;
	z-index:2;
	width:200px;
	height:100px;
	opacity:0.5;
}

#box2 {
	position:absolute;
	top:150px;
	left:150px;
	z-index:1;
	width:200px;
	height:100px;
	background-color:red;
}

#box3 {
	position:relative;
	width:300px;
	height:400px;
	background-color:green;
}


</style>
</head>
<body>

<div id="box3" class="box" >
	<div id="box1" class="box" >Hello World</div>
	<img id="image1" class="image">
	<div id="box2" class="box" ></div>
</div>

<script>
</script>
</body>
</html>
