Step:1

To add a sound to your game, start by adding this code below into your HTML section.


<audio id="coinAudio" src="./sounds/coin.mp3"></audio>

Step:2

To javascript section add this two functions to play and stop the coin sound.


function playCoinAudio() {
    var audio = document.getElementById('coinAudio');
    audio.play();
}

function pauseCoinAudio() {
    var audio = document.getElementById('coinAudio');
    audio.pause();
}

Step:3

And finally just add the play function to where ever in your javascript program you want to trigger the coin sound.

In my code, I added into the 'checkYellowCollision()' function.


function checkYellowCollision() {
    // Check collision with yellow squares
    for (var i = 0; i < yellowSquares.length; i++) {
        if (
            player.x < yellowSquares[i].x * tileSize + tileSize &&
            player.x + player.width > yellowSquares[i].x * tileSize &&
            player.y < yellowSquares[i].y * tileSize + tileSize &&
            player.y + player.height > yellowSquares[i].y * tileSize
        ) {
			playCoinAudio(); // Play Coin Audio
		}
	}
}

Adding Music

To add music to the game is the same as adding sound. Simple add the code below into your HTML section. Notice we add the word 'loop' at the end of the tag to indicate we want the music to play in a loop continously.


<audio id="musicAudio" src="./music/beeper-quest.mp3" loop></audio>
