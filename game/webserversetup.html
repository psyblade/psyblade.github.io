Webserver Setup

Step 1:

Download Simple Web Server and install the app.



Step 2:

After installing, run the app and click on the 'New Server' button to create new server.



Step 3:

Select your webfolder and note the port number



Step 4:

Once you have created a server you will see your server in the webserver's list



Step 5:

Open your web browser and navigate to http://127.0.0.1:[your_port_number]/
