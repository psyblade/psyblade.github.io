Lesson 39: Reusing the Star

If we want reuse this code, we have to put this code into a function and pass in the x, y coordinates:

function drawStar(x, y, color) {
	plot(x+1, y, color);
	plot(x-1, y, color);
	plot(x, y+1, color);
	plot(x, y-1, color);
}

The we call the function by writing:

drawStar(20, 20, "red");
drawStar(10, 10, "yellow");
