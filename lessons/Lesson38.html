Lesson 38: Draw a Star

We can hard code draw a Star given a coordinate point of (x=20, y=20) by writing:

plot(21, 20);
plot(19, 20);
plot(20, 21);
plot(20, 19);

But it's tedius if x and y changes for each star we want to draw. To make it easier do this:

var x = 20, y = 20;
plot(x+1, y, "cyan");
plot(x-1, y, "cyan");
plot(x, y+1, "cyan");
plot(x, y-1, "cyan");

Now simply change the value of x and y to draw your star at a different position.
