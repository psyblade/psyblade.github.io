Lesson 17: Random Color

We can also have the computer to pick a random color for us using the random() command, like this example below:

plot(random(0,59), random(0,39), random(0,15));

By specifying a range like (0,15), we are asking the computer to pick a color anywhere from 0 to 15 for us.
