Lesson 57: Nested Loops

A nested loop is a loop that exists within another loop.

For example, we can use a nested loop to fill the entire screen with a color using only the plot() function. 

for (let j = 0; j < 36; j++) {  // Outer loop
	for (let i = 0; i < 60; i++) {  // Inner loop
		plot(i, j, "lightblue");
  }
}

Outer Loop (for (let j = 0; j < 36; j++)): This loop runs 36 times, with j taking values from 0 to 35.

Inner Loop (for (let i = 0; i < 60; i++)): For each iteration of the outer loop, this loop runs 60 times, with i taking values from 0 to 59.
