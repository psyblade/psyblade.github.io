Lesson 31: Beeps

We can add beeps to our program by simply using the function 'beep' and passing in the notes which comprises the note value and the duration value.

	beep([[400,100], [500,100]]);
	beep ([[500, 100], [600, 100], [600, 100], [600, 100]]);
	beep ([[440, 100], [300, 100], [360, 100], [300, 100]]);
