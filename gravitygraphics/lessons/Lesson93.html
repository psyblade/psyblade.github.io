Lesson 93: For-each Loop

You can print an array using the for-loop...

let fruits = ["apple", "orange", "banana", "blueberry", "lime", "coconut"];
for(i=0; i < fruits.length; i++) {
	print(fruits[i])
}

... or you can print it using the for-each loop:

let fruits = ["apple", "orange", "banana", "blueberry", "lime", "coconut"];
fruits.forEach(fruit => {
  print(fruit);
});

Note that fruit (which is different from fruits which is an array) carries the current fruit that the for-each loop is pointing to.
