Lesson 39: Moving Animations

To do moving animations we simply draw something, delay for a bit, clear the canvas and do it all over again.

Now let's try to add clear canvas to our code and see what happens

for (i=0; i<100; i++) {
	plot(random(0,59), random(0,35), random(1,15));
	plot(random(0,59), random(0,35), random(1,15));
	await delay(1);
	clear();
}
