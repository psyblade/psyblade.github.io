Lesson 72-4: More Quadratic Equations

In this lesson we are going to show you how programs relate to the cartesian coordinates and graphing functions in math.

Here's how y = x3 + x2 + 2 look like in a program 

// y = x3 + x2 + 2

for (let x=-30; x < 60; x += 0.1) {
	y = x ** 3 + x ** 2 + 2;
	plotGraph(x, y);
}
