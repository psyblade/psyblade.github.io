Lesson 63: Keyboard Input: KeyDown

We can get a single key input from the user by detecting the key pressed. Here's how we do it:

// Function to handle key press
function onKeyDown(event) {
	text('Key pressed: ' + event.key)
}

// Add event listener to detect any key press
window.addEventListener('keydown', onKeyDown);

Run this code and then press any key on the keyboard to test it.
