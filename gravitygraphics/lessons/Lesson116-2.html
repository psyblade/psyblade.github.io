Lesson 116-2: Clock Face

To create a clock face we have to divide a circle into 12 equal parts. you need to calculate the angle for each part and then plot points along the circumference. Steps:

* Full Circle: The full circle is 2πr.
* Angle between each part: θ = 2 * Math.PI / numParts;
* Coordinates for each point: 
  x=r⋅cos⁡(θ)x=r⋅cos(θ)
  y=r⋅sin⁡(θ)y=r⋅sin(θ)
