Exercise: Change Size by Percentage

In the previous exercise we change the size of the rectangle using pixels. Now we are going to change the height using percentage.

If we want a size to be 50% bigger then we multiply the size by 1.5 (150/100). If we want the size to be 50% smaller, then we multiply by the size by 0.5 (50/100). The multiply sign is represented by the asterisk (*) 

normal_width = width * 1;
bigger_width = width * 2;
smaller_width = width * 0.5;
