Lesson 113: Array of Dictionaries

In the previous example, we use list/array for the planetary data. Now we are going to use dictionary/objects to hold our data. List/Arrays are declared with "= [ ]" and Dictionaries/Objects are declared with "= { }"

// Array of Arrays
var planets = [
	// [name, moons, sizeOnScreen]
	["Mercury", 0, 5],
	["Venus", 0, 12],
	["Earth", 1, 13],
	["Mars", 2, 7],
	["Jupiter", 79, 30],
	["Saturn", 83, 30],
	["Uranus", 27, 18],
	["Neptune", 14, 18]
];

// Array of Dictionaries
var planets = [
	{name: "Mercury", moons: 0, size: 5},
	{name: "Venus", moons: 0, size: 12},
	{name: "Earth", moons: 1, size: 13},
	{name: "Mars", moons: 2, size: 7},
	{name: "Jupiter", moons: 79, size: 30},
	{name: "Saturn", moons: 83, size: 30},
	{name: "Uranus", moons: 27, size: 18},
	{name: "Neptune", moons: 14, size: 18}
];

We can now access the name of the planet by just typing planets[0].name

// Array of Arrays
To access Earth’s name: planets[2][0] → "Earth"
To access Earth’s moons: planets[2][1] → 1

// Array of Dictionaries
To access Earth’s name: planets[2].name → "Earth"
To access Earth’s moons: planets[2].moons → 1

Or you can first transfer it to a variable planet:

// Array of Dictionaries
var planet = planets[2];
To access Earth’s name: planet.name → "Earth"
To access Earth’s moons: planet.moons → 1
