Lesson: Arrays

Arrays are ordered collections of values in JavaScript that can be accessed using numerical indexes. In other words, to print the fruit on a given shelf you would normally do:

var shelf = 2
if (shelf == 0) print ("apple");
if (shelf == 1) print ("orange");
if (shelf == 2) print ("banana");
if (shelf == 3) print ("blueberry");
if (shelf == 4) print ("lime");
if (shelf == 5) print ("coconut");

But by using array, you can make your code easier.

let array_fruits = ["apple", "orange", "banana", "blueberry", "lime", "coconut"];
print(array_fruits[2]);
