Lesson 72-2: Linear Function

Here's how y = 2x look like in a program, which is essentially the same as x = x + 1; and y = y + 2. 

// y = 2x
for (let x=-30; x < 30; x += 0.1) {
	y = 2 * x;
	plotGraph(x, y);
}

var x = 0;
var y = 0;
for (let i=0; i < 60; i += 0.1) {
	x = x + 0.1;
	y = y + 0.2;
	dot(x, 36 - y, 1, "cyan");
}
