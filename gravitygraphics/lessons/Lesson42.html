Lesson 42: Comparison Operators

Let's make a guessing game with comparison operators where the computer will hint the user to guess lower or higher until the user guess the correct number.

Comparison operators are used to compare two values and return a Boolean (true or false) based on the comparison.

< (less than): 5 < 10 // true
> (greater than): 10 > 5 // true
<= (less than or equal to): 5 <= 5 // true
>= (greater than or equal to): 10 >= 5 // true
== (equal to): 5 == 5 // true
!= (not equal to): 5 != 10 // true
=== (strict equal to): 5 === 5 // true
!== (strict not equal to): 5 !== '5' // true
