Lesson 71: (OOP) Object Literal

Object literal in JavaScript is a way to define an object using a set of key-value pairs directly within curly braces {}. It is one of the most straightforward methods for creating objects, and the keys (properties) can be assigned specific values.

Object literals are part of object-oriented programming (OOP) in JavaScript, but they are a more lightweight, simplified approach to defining objects. Using object literals, we will make the code from the previous lesson object oriented: 

const player = {
	x: 0,
	y: 20,
	speed: 0.3,
	
	movePlayer: function() {
		player.x += player.speed;
	},
	
	drawPlayer: function() { 
		plot(player.x, player.y, "blue"); 
	}
};
