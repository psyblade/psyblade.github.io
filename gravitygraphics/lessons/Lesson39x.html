Exercise: Animate the Border Drawing

Previously we tried to draw a border around the screen using the plot() function and For Loops. Now add in the delay() function to animate it.

for (i=0; i <= 60; i++) {
	plot(x=i, y=0, color="red");
}
for (i=0; i <= 60; i++) {
	plot(x=i, y=35, color="red");
}
for (i=0; i <= 60; i++) {
	plot(x=0, y=i, color="red");
}
for (i=0; i <= 60; i++) {
	plot(x=59, y=i, color="red");
}
