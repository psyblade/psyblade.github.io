Exercise: Draw 20 Random Color Stars

Now, let's make 20 random colored stars on the screen. For references the function to draw a star is:

function drawStar(x, y, color) {
	plot(x+1, y, color);
	plot(x-1, y, color);
	plot(x, y+1, color);
	plot(x, y-1, color);
}

And we call the function by writing:

drawStar(20, 20, "red");
