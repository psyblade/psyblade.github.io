Lesson 21: String

We call the word we printed, like "Hello" a string because it is made out of a string are characters.

Characters can be indivisual alphabets, letters, numbers or symbols.

For example printing "Hello":

print("Hello");

is the same as:

print("H" + "e" + "l" + "l" + "o");

We can join characters or strings using the plus + sign.
