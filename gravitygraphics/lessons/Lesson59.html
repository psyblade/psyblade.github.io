Lesson 59: Alternate Lines Using Mod

One of the uses of the mod operator is to generate alternate lines like the ones you see in your notebook.

The gist of it is:

for (let y=0; y < 40; y++) {
	result = y%2;
	if (result == 1) { // draw white line }
	if (result == 0) { // draw grey line }
}

To generate some gibberish text for testing, you can use the Lorem Ipsum function:

print (generateLoremIpsum(3)); // Generate a paragraph with 3 sentences
