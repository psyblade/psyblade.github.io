Lesson 49: Rounding Decimals

You can use rounding functions to handle the number of decimals you want to see. For example:

let num = 123.456789;

// Convert a value to a fixed number of decimal places
print(num.toFixed(2)); // Output: "123.46"

// parseInt(num) can convert a string or number to interger
print(parseInt(123.456789)); // Output: "123"
