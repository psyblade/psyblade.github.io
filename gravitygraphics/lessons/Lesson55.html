Lesson 55: Default Parameter Values

If we don't put any default values for x, y, width, height, color, then nothing will appear if we call the function myFrame();

But if we specify the default values of the parameters in the function such as shown below:

function myFrame(x=30, y=20, width=30, height=20, color=random(1,15)) {

}

then when we call the function myFrame(), a rectangle will appear based on the default parameter values of the function.
