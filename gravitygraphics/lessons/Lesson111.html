Lesson 111: Dictionary

In previous lesson we stored and access our Earth's data using arrays. Now we are going to use an object dictionary for storing the data for Earth. List / Arrays are declared with "= [ ]" and dictionaries are declared with "= { }"

// Data stored in array
// name, moons, size
var planet = ["Earth", 1, 10];

// Data stored in dictionary
var planet = {name: "Earth", moons: 1, size: 10};

To access the name of the planet by just typing planet.name

// Dictionary
To access Earth’s name: planet_earth.name → "Earth"
To access Earth’s moons: planet_earth.moons → 1

Dictionary makes it more readable, self descriptive and easier to understand.
