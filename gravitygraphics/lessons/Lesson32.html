Lesson 32: For Loop: Plotting 200 Dots

To plot 200 random dots we can't write the commond plot 200 times. That would be too tedious.

plot(random(0,60), random(0,39));
plot(random(0,60), random(0,39));
plot(random(0,60), random(0,39));
.
.
.

So what we can do instead, is use a "For Loop"

for (i = 0; i < 200; i++) {
	plot(random(0,60), random(0,39));
}
