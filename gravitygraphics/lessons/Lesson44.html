Lesson 44: Comments

As our code gets longer, a way to keep track of our code is to use comments. Comments makes it easier for other and ourselves to read what is written.

Use the double backstroke sign (//), to write a comment. Anything to the right of the (//) sign on the same line, the computer will ignore.

// Print week days
print("Monday");
print("Tuesday");
print("Wednesday");
print("Thursday");
print("Friday");

// Print weekends
print("Saturday");
print("Sunday");
