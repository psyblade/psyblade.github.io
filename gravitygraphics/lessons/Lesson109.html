Lesson 109: htab, vtab

We can specify where we want to print by setting the htab and vtab in the parameter of the print function.

We will use variables to store the data that we want to print.

var htab = 28;
var vtab = 4;

var planet_name = "Earth";
var planet_moons = 1;
var planet_size = 10;

print("Planet: " + planet_name, random(1,15), htab , vtab);
print("Moon: " + planet_moons, random(1,15), htab);
