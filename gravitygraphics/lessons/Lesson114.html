Lesson 114: Dictionary of Dictionaries

Instead of accessing our planet data using a number index i.e. planet[0], planet[1], we can also access it using a word index i.e. planet["Earth"], planet["Mars"]. In the previous example, we use an array (list) of objects (dictionary). Now we are going to use object (dictionary) of objects (dictionares) to hold our data.

// List of Dictionaries
var planets = [
	{name: "Mercury", moons: 0, size: 5},
	{name: "Venus", moons: 0, size: 12},
	{name: "Earth", moons: 1, size: 13},
	{name: "Mars", moons: 2, size: 7},
	{name: "Jupiter", moons: 79, size: 30},
	{name: "Saturn", moons: 83, size: 30},
	{name: "Uranus", moons: 27, size: 18},
	{name: "Neptune", moons: 14, size: 18}
];

// Dictionary of Dictionaries
var planets = {
    "Mercury": {name: "Mercury", moons: 0, size: 5},
    "Venus": {name: "Venus", moons: 0, size: 12},
    "Earth": {name: "Earth", moons: 1, size: 13},
    "Mars": {name: "Mars", moons: 2, size: 7},
    "Jupiter": {name: "Jupiter", moons: 79, size: 30},
    "Saturn": {name: "Saturn", moons: 83, size: 30},
    "Uranus": {name: "Uranus", moons: 27, size: 18},
    "Neptune": {name: "Neptune", moons: 14, size: 18}
};

We can now access the name of the planet by just typing planets["Earth"].name

// List of Dictionaries
To access Earth’s name: planets[2].name → "Earth"
To access Earth’s moons: planets[2].moons → 1

// Dictionary of Dictionaries
To access Earth’s name: planets["Earth"].name → "Earth"
To access Earth’s moons: planets["Earth"].moons → 1

Or you can first transfer it to a variable planet:

// Array of Dictionaries
var planet = planets["Earth"];
To access Earth’s name: planet.name → "Earth"
To access Earth’s moons: planet.moons → 1

This makes indexing easier, and more intuitive to find the planet you are looking for.
