Lesson 110: Planetary Info using Array

In previous lesson we stored and access our Earth's data using variables. Now we are going to use array for storing the data for Earth.

// Data stored in variables
var planet_name = "Earth";
var planet_moons = 1;
var planet_size = 10;

// Data stored in array
// name, moons, size
var planet = ["Earth", 1, 10];

We can now access the name of the planet by just typing planet[0]

// Array (List)
To access Earth’s name: planet[0] → "Earth"
To access Earth’s moons: planet[1] → 1
