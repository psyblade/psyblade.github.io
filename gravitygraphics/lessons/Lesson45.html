Lesson 45: Template Literals

Another syntax that we can use to enhance readability in our code is Template Literals.

For example, Instead of concatenating a string like:

const day = "Monday";
var message = "It's the start of the week!";
print("Today is " + day + ", " + message);

We can write this using template literals like this:

const day = "Monday";
var message = "It's the start of the week!";
print(`Today is ${day}, ${message}`);
