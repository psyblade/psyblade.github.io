Lesson 45a: Switch Case

There are certain syntax that we can use to enhance readability in our code.

For example, Instead of if-else statements like this:

const day = "Monday";
var message = "";

if (day === "Monday") {
  message = "It's the start of the week!";
} else if (day === "Friday") {
  message = "The weekend is almost here!";
} else {
  message = "It's just another day.";
}

We can write this using the switch-case statement like this:

const day = "Monday";
var message = "";

// Using a switch-case
switch(day) {
  case "Monday":
    message = "It's the start of the week!";
    break;
  case "Friday":
    message = "The weekend is almost here!";
    break;
  default:
    message = "It's just another day.";
}
