Lesson 41: Infitnite Loop

An infinite loop is a loop that keeps repeating itself. We can achieve this by setting the condition to always true. This for task like continuous animations.

while (true) {
	var i = 0;
	while (i < 36) {
		gr();
		line(0, i, 60, i);
		i = i + 1;
		await delay();
	}
	while (i > 0) {
		gr();
		line(0, i, 60, i);
		i = i - 1;
		await delay();
	}
}
