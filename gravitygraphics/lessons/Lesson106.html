Lesson 106: AABB Collision Detection

To add collision detection between the player (spaceship) and both asteroids and aliens, we can use axis-aligned bounding box (AABB) collision detection. This method checks if the bounding boxes of two objects overlap, which works well since both the spaceship, asteroids, and aliens are rectangular.

// AABB collision detection
function isColliding(obj1, obj2) {
    return obj1.x < obj2.x + obj2.width &&
           obj1.x + obj1.width > obj2.x &&
           obj1.y < obj2.y + obj2.height &&
           obj1.y + obj1.height > obj2.y;
}
