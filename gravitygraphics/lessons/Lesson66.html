Lesson 66: Slow Down the Frame Rate

One way to make it slower is to slow down the frame rate of the animation loop by using await delay()

To use await delay() in a function, we have to put "async" in front of the function like shown below:

// Function to update the pixel's position based on the current movement direction
async function updatePosition() {
	x += moveX;
	y += moveY;
	plot(x, y, color);  // Plot the pixel at the new position
	await delay(0.1);
	requestAnimationFrame(updatePosition);  // Call updatePosition again for the next frame
}
