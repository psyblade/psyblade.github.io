Exercise: Loading Images Into Memory

For images that we want to keep calling and reusing, we don't want to load it from the URL source everytime. We want to get the image from the URL and then load it into memory (or a variable) where we can quickly access it anytime. Loading it from memory (which is on your local machine) is much quicker than loading it from a URL source on the Internet. To load and draw an image into and from memory use this:

// Load image into memory
var image_holder = loadImageFromURL(src); 

// Draw image from memory
drawImage(img=image_holder, x=20, y=10, width=20, height=20); // Draw Image
