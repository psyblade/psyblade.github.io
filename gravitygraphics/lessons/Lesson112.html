Lesson 112: Array of Arrays

To add more planets, we can use an array of arrays to store the data.

var planets = [
	// [name, moons, sizeOnScreen]
	["Mercury", 0, 5],
	["Venus", 0, 12],
	["Earth", 1, 13],
	["Mars", 2, 7],
	["Jupiter", 79, 30],
	["Saturn", 83, 30],
	["Uranus", 27, 18],
	["Neptune", 14, 18]
];

To access the name of the planet by just typing planet[2].name

// Dictionary
To access Earth’s name: planets[2][0] → "Earth"
To access Earth’s moons: planets[2][1] → 1
