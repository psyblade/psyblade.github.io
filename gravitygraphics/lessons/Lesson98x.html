Exercise: Different Color Letters

You have the word "Hello" and you want to print "Hel" in red and "lo" in green. The way to do it is:

var x_center = 30, y_center = 20;
var spacing = 2, word = "Hello";
for (let i = 0; i < word.length; i++) {
	text(word[i], "red", ((x_center - (word.length/2)) + (i * spacing)), y_center); 
}
for (let i = 3; i < word.length; i++) {
	text(word[i], "green", ((x_center - (word.length/2)) + (i * spacing)), y_center);  
}

(word.length/2) is to center the word by dividing the width of the word into 2
