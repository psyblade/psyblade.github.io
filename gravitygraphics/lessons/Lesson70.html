Lesson 70: Object Oriented Programming

Object-oriented programming (OOP) is grouping variables (data) and functions (behavior) around the concept of "objects,", making it easier to manage complex systems.

For example, instead of variables playerX, playerY, speed, and the functions movePlayer(), and drawPlayer() which are scattered throughout the code, we want to group them together as an object. One way we can do that is using Object Literals. Have a look at how the our code looks below without Object Literal. Now proceed to the next lesson to see how we can use Object Literals to organize our code. 

var playerX = 0;
var playerY = 20;
var playerSpeed = 0.3;

function movePlayer() { 
	playerX += playerSpeed;
}

function drawPlayer() { 
	plot(playerX, playerY, "blue"); 
}
