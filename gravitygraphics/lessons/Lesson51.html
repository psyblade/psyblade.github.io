Lesson 51: Functions

Functions is a way to group and organize your code to make it neat, simple, and more readable. For example instead of just generating a random percentage like this:

var number = random(1,100);
text(number + "% = " + number + "/100 = " + (number / 100));

We can write a randomPercentage() function for more readability:

function randomPercentage() {
	var number = random(1,100);
	text(number + "% = " + number + "/100 = " + (number / 100));
}
randomPercentage();

After writing the function we have to call the function to execute it. Otherwise, it would not do anything.

This is a function without parameters.
